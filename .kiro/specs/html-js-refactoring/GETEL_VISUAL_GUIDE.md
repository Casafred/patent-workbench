## ðŸŽ¯ Key Concepts

### Why Conditional Didn't Work
```javascript
// âŒ This DOESN'T work:
if (typeof getEl === 'undefined') {
    var getEl = (id) => document.getElementById(id);
}

// Why? JavaScript hoisting:
// 1. Parser phase: Sees "var getEl" â†’ hoists to top
// 2. Parser phase: Sees "const getEl" from tab-navigation.js
// 3. Parser phase: CONFLICT! Two declarations in same scope
// 4. Runtime phase: Never reached, error thrown during parsing
```

### Why Removal Works
```javascript
// âœ… This DOES work:
// (in tab-navigation.js)
const getEl = (id) => document.getElementById(id);

// (in main.js)
// No definition needed - just use it!
getEl('my-id').classList.add('active');

// Why? Single source of truth:
// 1. Parser phase: Only ONE "const getEl" declaration
// 2. Parser phase: No conflicts
// 3. Runtime phase: getEl is available globally
// 4. Runtime phase: All scripts can use it
```
