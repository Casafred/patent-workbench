# 快速测试 v4 - 一分钟验证

## ⚡ 第一步：清除缓存（必须！）

**Windows**: `Ctrl + Shift + R`  
**Mac**: `Cmd + Shift + R`

## 🎯 第二步：快速测试（3分钟）

### 1. 悬浮球拖动（30秒）
1. 找到右下角绿色❓按钮
2. 拖动它到屏幕左上角
3. **检查**: 拖动流畅？有阴影效果？
4. 点击按钮
5. **检查**: 正常打开帮助页面？

✅ **通过**: 拖动流畅，点击正常  
❌ **失败**: 拖动卡顿或点击无反应

### 2. 模型选择器（1分钟）
1. 打开页面，等待2秒
2. 按`F12`打开控制台
3. **检查**: 看到`✅ 所有模型选择器已更新`？
4. 点击"功能二：小批量异步"
5. 点击"选择预置模板或新建"
6. 选择"技术文本翻译"
7. **检查**: "模型"下拉框有选项？

✅ **通过**: 所有下拉框都有模型列表  
❌ **失败**: 下拉框是空的

### 3. 模板选择器（30秒）
1. 在"功能二"页面
2. **检查**: "选择预置模板或新建"有3个选项？
   - 技术文本翻译
   - 检索词拓展
   - 技术文本总结

✅ **通过**: 看到3个预置模板  
❌ **失败**: 下拉框是空的或只有"选择预置模板或新建"

## 📊 控制台检查（30秒）

按`F12`，应该看到：
```
✅ 模型配置已从 config/models.json 加载
✅ 功能一模型选择器已更新
✅ 功能二模型选择器已更新

```
