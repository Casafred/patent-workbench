Abstract

公开了一种凸轮轴，所述凸轮轴具有不对称倒角，所述不对称倒角用于在施加即时的减速力时引导滚珠轴承更深地进入凸轮轴凹槽以避免滚珠的遗失。当具有凸轮轴和锤的工具掉落时，倒角将滚珠与相对于凸轮轴轴向移动的锤的前表面的开口隔开。因此，当冲击扳手掉落时，滚珠从锤的前开口滑落的可能性降低。

Description

滚珠偏转倒角
技术领域
本申请涉及滚珠轴承系统。特别地，本申请涉及滚珠偏转倒角，当向滚珠轴承施加力时，该滚珠偏转倒角使滚珠轴承转向至有利位置，以防止滚珠从系统的意外脱离。
背景技术
许多工具具有通过滚珠轴承彼此相互作用的部件。滚珠通常设置在一个槽或圆形孔洞内，在两个移动部件之间提供滚动连接，减少部件之间的摩擦力且能实现部件之间的相对运动。
例如，冲击扳手包括在凸轮轴和锤之间的滚珠轴承接口。滚珠被插入沿着凸轮轴的外围设定的直线的或者弯曲的凹槽，并与锤中的配合凹槽接合。图5所示的为这样一种凸轮轴。凸轮轴535包括基座540和自所述基座540延伸的轴545。轴545包括用于容纳与锤中的凹槽接合的滚珠轴承的凹槽550。凸轮轴535也包括孔洞555，该孔洞555用于容纳与其他部件连接的紧固件，例如螺丝或钉子。
现有的凸轮轴535包括具有顶部560的凹槽。当冲击扳手掉落时，一旦凸轮轴535撞击地面时，凸轮轴535会突然停止，滚珠在减速力的作用下移动至凸轮轴535的顶部560。同时，锤可移动地设置在轴545上，并轴向移动对抗弹簧，滑过滚珠轴承并通过在锤前侧的开口使滚珠轴承脱离，进而损坏滚珠轴承系统。
发明内容
本申请公开了一种用于引导滚珠轴承更深地进入凹槽以避免滚珠丢失的结构。本发明的凸轮轴包括相交于一顶部、并且靠近所述顶部的凹槽，不对称倒角用于在例如工具不慎掉落时引起的减速力期间引导滚珠。倒角引导滚珠更深地进入凹槽，以使得滚珠与锤的前表面的开口隔开。因此，当冲击扳手掉落时，滚珠从锤前表面的开口掉落的可能性较低。
例如，本申请公开了一种用于在工具中使用的凸轮轴，所述凸轮轴包括：基座；自所述基座在轴向方向上延伸的轴；围绕所述轴圆周地设置并用于可移动地挡住滚珠轴承的凸轮轴凹槽，所述凸轮轴凹槽包括以相对于轴向方向成一定角度延伸并相交于顶部的多个凹槽部分；倒角，其限定于所述凸轮轴凹槽内并用于当向所述凸轮轴施加轴向方向的力时使滚珠远离顶部并朝基座偏斜。
也公开了一种组件，该组件包括用于在工具中使用的凸轮轴，该凸轮轴包括：基座；自所述基座在轴向方向上延伸的轴；围绕所述轴圆周地设置并用于可移动地挡住滚珠轴承的凸轮轴凹槽，所述凸轮轴凹槽包括两个凹槽部分，所述凹槽部分以相对于轴向方向成一定角度延伸并相交于顶部；倒角，其限定于所述凸轮轴内并用于当向所述凸轮轴施加轴向方向的力时使滚珠远离顶部并朝基座偏斜；锤，其具有开口，该开口设置在所述锤中以用于容纳滚珠，所述开口延伸穿过锤至锤的前表面；以及偏压机构，其设置在所述凸轮轴和所述锤之间。
进一步公开了一种工具，该工具包括：用于向工件提供动力的动力源；用于从动力源选择性地分配动力的触发器；用于接受动力并向工件施加冲击力和/或扭矩的冲击端；凸轮轴，该凸轮轴包括：基座；自所述基座在轴向方向上延伸的轴；围绕所述轴圆周地设置并用于可移动地挡住滚珠轴承的凸轮轴凹槽，所述凸轮轴凹槽包括两个凹槽部分，所述凹槽部分以相对于轴向方向成一定角度延伸并相交于顶部；倒角，其限定于所述凸轮轴内并用于当向所述凸轮轴施加轴向方向的力时使滚珠远离顶部并朝基座偏斜；锤，其具有开口，该开口设置在所述锤中以用于容纳滚珠，所述开口延伸穿过锤至锤的前表面；以及偏压机构，其设置在所述凸轮轴和所述锤之间。

本发明公开了一种凸轮轴，该凸轮轴具有不对称倒角，该不对称倒角用于在向凸轮轴施加减速力时引导滚珠轴承进入凸轮轴凹槽的更深处进而避免滚珠的丢失。倒角偏离滚珠在锤的轴向运动中更深地进入凹槽并远离锤的前开口。因此，当工具，例如冲击扳手在意外掉落时，滚珠从锤的前表面滑落的可能性降低。
图1所示的为本发明一实施例的工具100。如图所示，工具100为冲击扳手。可以理解的是，在不脱离本发明保护范围和构思的前提下，本发明可以和其他工具或设备一起使用。在一实施例中，工具100包括冲击端105，该冲击端105用于和动力源110，如电池，燃料电池或气动源连接。当用户拉动触发器115时，动力源110向冲击端105施加动力。换向杆120可设置在工具上以反转冲击端105的驱动方向，通风口125可形成在工具100的腔室内，将多余的热量从工具100中排出。工具100也可以包括在使用期间用于使用户握紧工具100的夹子130。
动力源110可以是任意一种能够给工具100提供动力的电动或气动源。在一实施例中，动力源110为电池。然而，动力源110可以是提供动力的任何组件，包括电池，燃料电池，发动机，太阳能发电系统，风力发电系统，水力发电系统，用于与插座连接的电源线，或者任意一种提供动力的方式。
图2所示的为本发明一实施例组件132的分解图以及图3为组件132的侧剖视图。如图所示，组件132包括凸轮轴135，该凸轮轴135具有基座140和自基座沿轴向方向延伸的轴145。轴145包括凸轮轴凹槽150，该凸轮轴凹槽150围绕轴145圆周地延伸并用于承托滚珠轴承。基座155可以包括孔洞，该孔洞设置在所述基座150中用于容纳紧固件以及用于凸轮轴135和工具100其他组件的组装。
如图所示，凸轮轴145可以抵靠偏压机构160，所述偏压机构160与垫圈165和轴承170对齐。一个或多个滚珠175可以设置在凸轮轴凹槽150内部并用锤180接合凸轮轴145。锤180可以包括锤凸耳185和锤凹槽190，该锤凸耳185用于接收径向冲击，锤凹槽190用于容纳滚珠175。可替代地，锤180可以包括位于锤180的前表面的开口，其未被分成多个独立的凹槽。锤凹槽190也可以部分地穿过锤180且不延伸至前表面180a。
砧座195可以相对于锤180轴向设置并通过砧座凸耳200向锤180施加冲击力。在操作过程中，锤180可以接收来自动力源110的动力并连续旋转，也就是说，时有时无的冲击而不是连续的旋转速度或扭矩。当锤凸耳185撞击砧座凸耳200时，锤180可以冲击砧座195，使砧座195旋转并与冲击端105共同作用于工件。锤180也可以基于电池110供应的动力和砧座195提供的冲击而轴向移动，从而压缩偏压机构160并沿凸轮轴135的轴145轴向移动。
当工具100掉落时，其导致瞬间的减速力施加到工具100上，滚珠175可以移动到最接近锤180前表面180a的凸轮轴凹槽150的顶点处，并且凸轮轴135保持静止。同时，锤180沿轴145轴向移动并向偏压机构160施加比在正常操作期间由于工具100掉落所受到的力大的力。所以增加的力比正常操作期间更多地压缩偏压机构160，从而使锤180更深地位于轴145上，即更靠近基座140。现有技术中，锤的前表面将设置在比位于凸轮轴凹槽内的滚珠更靠近凸轮轴的基座的位置处，从而将滚珠置于从锤前表面处的凸轮轴凹槽中滑出的危险。然而，如下所述，本申请公开的改进提供了一种位于凸轮轴上、用于引导滚珠进入滑落可能性较低的位置的倒角。
如图4所示，区别于图5，与图5中所示的现有技术相比，凸轮轴135可以包括靠近顶部210的倒角205以将滚珠175偏移到凸轮轴凹槽150的更深处。如图所示，凸轮轴凹槽150可以包括两个凹槽部分，该凹槽部分相对于凸轮轴135的轴线方向呈一定的角度并相交于顶部210。在操作过程中，如果工具100掉落，滚珠175偏离至呈角度的凸轮轴凹槽210的更深处并远离锤180的前表面180a。这样，当工具掉落时，滚珠175从锤180前表面180a滑落的可能性降低。
