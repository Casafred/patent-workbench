🧪 功能六增强功能测试指南

⚠️ 重要提示：

这些新功能是在你的应用中实现的，不是在Google Patents网站上！

请确保访问：http://43.99.101.195

📋 测试前准备

  1. 确认服务器已更新（执行 update_aliyun_now.bat）
  2. 访问 http://43.99.101.195
  3. 使用你的账号登录
  4. 点击"功能六：批量专利解读"标签

🎯 测试项目清单

测试1：基本查询（不勾选说明书）

  • 输入专利号：US10000000B2
  • 不勾选"获取说明书字段"
  • 点击"批量查询专利"
  • 等待查询完成（约5-10秒）
  • 检查是否显示基本信息
  • 检查每个字段旁边是否有📋按钮
  • 点击摘要旁的📋按钮，测试复制功能
  • 检查是否显示权利要求
  • 检查权利要求旁是否有📋按钮
  • 检查是否显示附图（如果有）

测试2：完整查询（勾选说明书）

  • 输入专利号：US10000000B2
  • 勾选"获取说明书字段"
  • 点击"批量查询专利"
  • 等待查询完成（约10-20秒，因为要获取更多信息）
  • 向下滚动，检查是否显示说明书
  • 继续向下滚动，检查是否显示绿色区域的"引用专利"表格
  • 检查是否显示橙色区域的"被引用专利"表格
  • 检查是否显示紫色区域的"法律事件"表格
  • 测试每个进阶信息的📋复制按钮

测试3：字段复制功能

  • 点击任意字段的📋按钮
  • 按钮应该变成✓
  • 打开记事本，按Ctrl+V粘贴
  • 验证内容是否正确
  • 测试至少3个不同字段的复制

测试4：附图显示

  • 查看是否有"🖼️ 专利附图"区域
  • 检查附图是否正常显示
  • 点击附图，应该在新窗口打开大图
  • 如果图片加载失败，应该显示"图片加载失败"

🔍 故障排查

问题1：没有看到新功能

可能原因：
  • 服务器代码未更新
  • 浏览器缓存未清除
  • 访问了错误的URL
解决方案：
  1. 按 Ctrl+Shift+R 强制刷新页面
  2. 清除浏览器缓存
  3. 确认访问的是 http://43.99.101.195
  4. 检查服务器是否已更新（SSH登录查看）

问题2：附图不显示

可能原因：
  • 该专利本身没有附图
  • Google Patents页面结构变化
  • 网络问题
解决方案：
  1. 尝试其他专利号（如 CN104154208B）
  2. 打开浏览器开发者工具（F12）查看Network标签
  3. 查看服务器日志

问题3：进阶信息不显示

⚠️ 重要：

进阶信息（引用专利、被引用专利、法律事件）只有勾选"获取说明书字段"时才会提取！

如果不勾选，这些信息不会显示。
