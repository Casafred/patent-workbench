# 功能七：权利要求处理

## 功能简介

权利要求处理功能是专利智能工作台专为专利权利要求分析设计的核心功能之一。该功能支持Excel批量分析和文本输入分析两种方式，能够自动识别独立权利要求和从属权利要求，并生成可视化的权利要求引用关系图，帮助用户深入理解专利权利要求的结构和关系。

## 核心特性

### 1. Excel批量分析
- **Excel文件上传**：支持上传包含专利权利要求的Excel文件
- **工作表选择**：可选择要处理的工作表
- **权利要求列选择**：可指定包含权利要求的列
- **自动分类**：自动识别独立权利要求和从属权利要求
- **语言识别**：支持中英文混合文本
- **引用关系提取**：自动提取权利要求之间的引用关系
- **置信度评分**：为分析结果提供置信度评分
- **结果导出**：支持导出为Excel或JSON格式
- **公开号与独权合并**：显示每个公开号及其对应的独立权利要求合并内容

### 2. 文本输入分析
- **直接文本输入**：支持直接输入权利要求文本
- **示例加载**：提供示例权利要求，便于用户理解和使用
- **自动分类**：自动识别独立权利要求（蓝色高亮）和从属权利要求（灰色显示）
- **实时分析**：输入后即可进行分析，速度快
- **结果可视化**：直观展示分析结果，包括统计信息和权利要求列表

### 3. 权利要求引用关系图生成
- **多种可视化样式**：支持树状图、网络图、径向图三种样式
- **交互操作**：支持缩放、居中、重置、高清截图等操作
- **散开程度调整**：支持调整树状图的散开程度
- **高清截图**：支持生成高清截图，方便用于专利文档
- **动态生成**：根据权利要求自动生成引用关系图
- **多专利号支持**：支持为多个专利号生成引用关系图

### 4. 智能分析
- **特征提取**：自动提取权利要求的核心特征
- **关系分析**：分析权利要求之间的引用关系
- **统计信息**：提供权利要求数量、独立权利要求数量、从属权利要求数量等统计信息
- **置信度评估**：为分析结果提供置信度评分，便于用户判断结果可靠性

## 操作步骤

### 1. Excel批量分析
- **步骤1**：点击"Excel批量分析"标签
- **步骤2**：上传包含专利权利要求的Excel文件
- **步骤3**：选择工作表和权利要求列
- **步骤4**：点击"开始处理"按钮
- **步骤5**：查看处理进度和结果
- **步骤6**：可选择导出为Excel或JSON格式
- **步骤7**：在"公开号与独权合并内容"区域，可查看每个公开号及其对应的独立权利要求合并内容
- **步骤8**：点击"生成权利要求关系图"按钮，可生成可视化的权利要求引用关系图
- **步骤9**：可选择不同的可视化样式，调整散开程度，进行缩放、居中、截图等操作

### 2. 文本输入分析
- **步骤1**：点击"文本输入分析"标签
- **步骤2**：在文本框中输入权利要求文本，或点击"加载示例"按钮加载示例
- **步骤3**：点击"开始分析"按钮
- **步骤4**：查看分析结果，包括统计信息和权利要求列表
- **步骤5**：在"权利要求引证关系图"区域，可选择不同的可视化样式
- **步骤6**：可调整散开程度，进行缩放、居中、截图等操作
- **步骤7**：点击"清空"按钮可清空输入和结果

## 注意事项

1. Excel文件应包含清晰的权利要求文本，避免格式混乱
2. 文本输入时，权利要求应符合规范格式，包括编号和特征描述
3. 生成的可视化图可能因权利要求数量和复杂度而有所不同
4. 置信度评分仅供参考，最终结论需结合专业知识判断
5. 处理大量权利要求时，可能需要较长时间，请耐心等待
6. 建议定期保存分析结果，避免数据丢失
