# 功能五：权利要求对比

## 功能简介

权利要求对比功能是专利智能工作台专为专利对比分析设计的核心功能之一。该功能支持2-4个权利要求的智能对比，自动识别语言、按需翻译，提供多种可视化展示方式，能够帮助用户快速分析专利权利要求之间的异同。

## 核心特性

### 1. 智能对比系统
- **对比数量支持**：支持2-4个权利要求同时对比
- **模型选择**：支持选择不同AI模型（GLM-4.7-Flash、glm-4-flash、glm-4-long）
- **自动语言识别**：支持自动识别权利要求语言
- **按需翻译**：支持将权利要求翻译为中文或英文
- **差异统计**：自动统计相同特征、差异特征和平均相似度

### 2. 多视图可视化
- **卡片视图**：以卡片形式展示每个权利要求及其特征
- **并排对比**：将权利要求并排展示，便于逐句对比
- **矩阵视图**：以矩阵形式展示权利要求之间的相似度
- **差异高亮**：自动高亮显示差异特征，便于识别

### 3. 结果管理与导出
- **结果预览**：直观展示对比结果，包括相同特征和差异特征
- **报告导出**：支持导出对比报告，方便分享和存档
- **语言切换**：支持切换为原文或译文，便于理解

### 4. 智能分析
- **特征提取**：自动提取权利要求的核心特征
- **相似度计算**：计算权利要求之间的相似度
- **差异分析**：分析权利要求之间的差异，包括新增特征和缺失特征

## 专利领域优势

### 对比市面AI工具的优势
1. **权利要求结构理解**：专门针对权利要求结构设计，能够准确理解独立权利要求和从属权利要求
2. **精细差异分析**：能够识别权利要求之间的细微差异，包括技术特征的增减和修改
3. **多种可视化方式**：提供卡片、并排、矩阵三种视图，适应不同对比需求
4. **多语言支持**：支持自动识别和翻译多种语言的权利要求
5. **全面的差异统计**：自动统计相同特征、差异特征和平均相似度，便于量化分析

### 针对专利工作特点的优化
- **精细工作**：能够识别权利要求中的细微差异，支持逐句对比
- **严谨工作**：提供准确的相似度计算和差异分析，支持量化评估
- **文本处理**：能够处理长文本权利要求，支持多种语言

## 使用场景

1. **专利审查意见答复**：对比本申请权利要求与现有技术，分析区别特征
2. **无效宣告证据分析**：分析无效宣告请求中的证据与目标专利的关系
3. **专利侵权比对**：对比涉嫌侵权产品与专利权利要求，分析侵权可能性
4. **专利撰写质量评估**：评估权利要求之间的逻辑关系和保护范围
5. **现有技术检索辅助**：对比检索到的现有技术与本申请，分析新颖性和创造性

## 操作步骤

### 1. 输入权利要求
- 在权利要求输入区域输入2-4个要对比的权利要求
- 系统会自动识别权利要求的语言

### 2. 配置对比参数
- 选择对比模型（GLM-4.7-Flash、glm-4-flash、glm-4-long）
- 选择对比数量（2-4个）

### 3. 开始分析
- 点击"开始分析"按钮
- 系统会自动处理权利要求，提取特征并进行对比

### 4. 查看结果
- 在结果展示区域查看对比结果
- 可切换不同视图（卡片、并排、矩阵）
- 可查看差异统计信息
- 可切换为原文或译文

### 5. 导出报告
- 点击"导出报告"按钮
- 系统会生成对比报告，可下载保存

2. **企业IPR**：用于专利侵权风险评估、竞争对手专利分析等
3. **专利审查员**：用于辅助专利审查，分析权利要求的新颖性和创造性
4. **研发人员**：用于评估研发成果的专利保护范围
5. **专利学习者**：用于学习权利要求对比分析方法

## 功能亮点

- **智能对比算法**：能够准确识别权利要求之间的异同
- **多种可视化方式**：提供三种视图，适应不同对比需求
- **全面的差异分析**：自动统计相同特征、差异特征和平均相似度
- **多语言支持**：支持自动识别和翻译多种语言的权利要求
- **一键导出报告**：便于分享和存档

## 注意事项

1. 输入的权利要求应符合专利格式规范，包括编号和特征描述
2. 对比结果仅供参考，最终结论需结合专业知识判断
3. 选择合适的对比模型，平衡处理速度和分析深度
4. 对于复杂的权利要求，建议使用并排视图进行逐句对比
5. 导出的报告应结合原始权利要求进行验证
6. 系统支持的语言有限，对于少见语言可能无法准确识别和翻译
