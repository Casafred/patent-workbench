# 功能一：即时对话

## 功能简介

即时对话是专利智能工作台的核心功能之一，为专利领域用户提供专业的AI对话服务。该功能支持多种交互方式，包括文本对话、文件上传和联网搜索，能够适应专利工作中的各种场景需求。

## 核心特性

### 1. 智能对话系统
- 支持多种AI模型选择，可根据需求调整温度参数
- 提供上下文管理功能，可配置上下文条数
- 支持对话标题编辑和历史对话管理

### 2. 自定义角色管理
- 可创建、编辑和删除自定义对话角色
- 支持为角色配置系统提示和用户模板
- 支持角色预设，适应不同专利场景

### 3. 文件上传与管理
- 支持多种文件格式上传（PDF、Word、Excel、图片、CSV等）
- 提供文件管理功能，可查看和管理已上传文件
- 支持按用途筛选文件（内容抽取、agent、代码解释器、批量处理）

### 4. 联网搜索功能
- 集成智谱网络搜索API，支持实时获取最新信息
- 搜索结果可直接用于对话上下文

### 5. 对话历史与导出
- 自动保存对话历史，支持切换和管理多个对话
- 支持多种导出格式：TXT、PNG、PDF
- 支持消息管理，可选择删除部分或全部消息

## 专利领域优势

### 对比市面AI工具的优势
1. **专利专业知识支持**：专门针对专利领域优化，能够准确理解专利术语和概念，避免泛泛而谈
2. **多模态交互**：支持文件上传和联网搜索，能够处理专利文档和获取最新专利信息
3. **灵活的上下文管理**：适合深入分析复杂专利问题，能够保持对话的连贯性
4. **自定义角色系统**：可根据不同专利工作场景（审查、无效、撰写等）创建专属角色
5. **安全可靠**：本地数据处理，保护专利敏感信息

### 针对专利工作特点的优化
- **精细工作**：支持逐句分析专利文本，能够处理权利要求、说明书等复杂内容
- **严谨工作**：提供准确的专利相关问答，支持引用外部信息
- **文本处理**：能够处理长文本专利文档，支持多种文件格式

## 使用场景

1. **专利技术咨询**：询问专利相关技术问题，获取专业解答
2. **法律问题解答**：咨询专利法相关问题，了解专利审查规则
3. **技术特征分析**：分析专利技术特征，提取核心创新点
4. **现有技术检索辅助**：结合联网搜索，查找相关现有技术
5. **专利文档解读**：上传专利文档，获取AI解读和分析

## 操作步骤

### 1. 开启新对话
- 点击"历史对话"面板中的"+"按钮，或在当前对话标题栏点击"开启新对话"
- 系统会创建一个新的未命名对话

### 2. 配置对话参数
- 点击"对话参数设置"按钮
- 选择合适的AI模型和温度参数
- 配置上下文条数
- 选择或创建自定义角色

### 3. 进行对话
- 在输入框中输入问题或指令
- 可点击文件上传按钮上传专利文档
- 可点击搜索按钮进行联网搜索
- 点击"发送"按钮提交请求

### 4. 管理对话
- 可编辑对话标题
- 可在"历史对话"面板中切换不同对话
- 可点击"管理消息"进行消息管理

### 5. 导出对话
- 点击"导出"按钮，选择导出格式（TXT、PNG、PDF）
- 系统会生成对应格式的文件并下载

## 适用人群

1. **专利代理人/律师**：用于专利撰写、审查意见答复、无效宣告等工作
2. **企业IPR**：用于专利检索、技术分析、竞争对手监控等
3. **研发人员**：用于技术调研、现有技术分析、创新方向探索等
4. **专利审查员**：用于辅助专利审查，提高审查效率
5. **专利学习者**：用于学习专利知识，了解专利领域最新动态

## 功能亮点

- **灵活配置**：支持多种模型和参数配置，适应不同需求
- **多模态支持**：文本、文件、联网搜索相结合，提供全面的信息支持
- **自定义角色**：可根据不同场景创建专属角色，提高工作效率
- **便捷导出**：支持多种导出格式，方便分享和存档
- **用户友好界面**：直观的对话界面，易于操作和管理

## 注意事项

1. 上传的专利文件应确保合法性和授权
2. 联网搜索结果仅供参考，需自行验证准确性
3. 对话历史会保存在本地浏览器中，建议定期导出重要对话
4. 可根据需求调整上下文条数，平衡对话连贯性和API调用成本
