# 功能四：本地专利库管理

## 功能简介

本地专利库管理功能是专利智能工作台专为专利数据管理设计的核心功能之一。该功能支持同族专利号展开、新旧库合并和自定义匹配规则，能够帮助用户高效管理和更新本地专利库。

## 核心特性

### 1. 同族专利号展开
- **Excel文件上传**：支持上传包含同族专利号的Excel文件
- **工作表选择**：可选择要处理的工作表
- **忽略首行选项**：支持忽略无效的首行信息
- **同族专利号列配置**：可指定包含同族专利号的列名
- **分隔符配置**：支持自定义同族专利号的分隔符
- **去重处理**：自动去除重复的专利号
- **结果预览**：直观展示展开后的专利号列表
- **结果导出**：支持导出为TXT格式，方便后续使用

### 2. 新旧库合并
- **原始库与新库上传**：支持上传原始专利库和新专利库
- **多列匹配**：支持配置多个匹配列，确保数据准确性
- **列选择**：可选择需要从旧表补充到新表的列
- **匹配规则配置**：可配置用于匹配的列（通常是公开号）
- **智能合并**：自动合并匹配的数据，保留最新信息
- **结果预览**：可视化展示合并结果，支持查看前10行数据
- **最终结果导出**：支持导出为Excel格式，方便使用

### 3. 智能数据处理
- **自动列识别**：支持自动识别Excel中的数据列
- **数据清洗**：自动处理无效数据，提高数据质量
- **可视化操作**：直观的操作界面，易于使用
- **进度提示**：实时显示处理进度，便于跟踪

## 操作步骤

### 1. 上传与展开
- **步骤1**：上传原始专利Excel文件
- **步骤2**：选择工作表，设置是否忽略首行
- **步骤3**：配置包含同族专利号的列名和分隔符
- **步骤4**：点击"展开同族专利号"按钮
- **步骤5**：查看展开结果，可复制或下载为TXT文件

### 2. 合并新旧库
- **步骤1**：加载原始专利库（从上一步或直接上传）
- **步骤2**：上传新专利库
- **步骤3**：选择要从旧表补充到新表的列
- **步骤4**：配置用于匹配的列（通常是公开号）
- **步骤5**：点击"开始合并与更新"按钮
- **步骤6**：查看合并状态和结果

### 3. 下载最终结果
- **步骤1**：查看合并结果预览
- **步骤2**：确认无误后，点击"下载最终合并的Excel文件"按钮
- **步骤3**：保存下载的Excel文件

## 注意事项

1. Excel文件大小建议不超过50MB，避免影响处理速度
2. 建议将同族专利号放在单独的列中，便于系统识别
3. 配置匹配列时，建议选择唯一标识符（如公开号），确保匹配准确性
4. 合并操作会覆盖原文件，建议备份重要数据
5. Excel中的图片、图表等对象不会被保留在最终导出的文件中
6. 处理大量数据时，可能需要较长时间，请耐心等待
7. 建议定期备份专利库数据，避免数据丢失
