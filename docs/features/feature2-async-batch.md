# 功能二：小批量异步

## 功能简介

小批量异步功能是专利智能工作台专为处理中等规模专利数据设计的核心功能之一。该功能支持Excel导入和手动添加两种输入方式，提供灵活的模板配置系统，能够高效处理小批量专利数据，并生成结构化结果。

## 核心特性

### 1. 灵活的输入方式
- **Excel导入**：支持上传Excel文件，自动识别工作表和数据列
- **手动添加**：支持直接输入数据，每行一条，适合少量数据
- **多列数据支持**：可配置1-10列数据，适应复杂专利数据结构

### 2. 自定义模板系统
- **模板创建**：可创建和保存自定义处理模板
- **模型配置**：支持选择AI模型和调整温度参数
- **系统提示和用户模板**：可配置系统提示和用户模板，指导AI生成结果
- **输出格式定义**：支持自定义JSON输出格式，确保结果结构化

### 3. 批量请求管理
- **请求范围配置**：可为每个模板指定要处理的输入数据序号范围
- **请求预览**：支持预览待提交的请求列表
- **任务状态跟踪**：实时显示任务状态，包括成功、失败、处理中
- **结果导出**：支持导出为Excel格式，方便进一步分析

### 4. 智能数据处理
- **自动列识别**：支持自动识别Excel中的数据列
- **数据批量处理**：高效处理小批量数据，提高工作效率
- **结果可视化**：直观展示处理结果，支持查看详细信息

## 专利领域优势

### 对比市面AI工具的优势
1. **专利数据结构优化**：专门针对专利数据结构设计，支持多列关联分析
2. **可复用模板系统**：创建的模板可重复使用，适合重复的专利分析任务
3. **灵活的请求配置**：可根据需求配置处理范围，避免不必要的API调用
4. **结构化结果输出**：支持自定义JSON输出格式，便于后续数据处理
5. **直观的任务管理**：可视化任务状态，便于跟踪和管理

### 针对专利工作特点的优化
- **精细工作**：支持逐行处理专利数据，能够提取权利要求特征、技术要点等详细信息
- **严谨工作**：提供结构化的结果输出，确保数据准确性和一致性
- **文本处理**：能够处理多种格式的专利文本数据，支持中英文混合文本

## 使用场景

1. **小批量专利技术分析**：对10-50个专利进行技术特征提取和分析
2. **权利要求特征提取**：批量提取专利权利要求的核心特征
3. **申请人/发明人关联分析**：分析专利申请人和发明人之间的关联关系
4. **专利分类整理**：根据技术特征对专利进行分类整理
5. **专利态势初步分析**：对特定技术领域的专利进行初步分析

## 操作步骤

### 1. 输入内容
#### 方式一：Excel导入
- 点击"Excel导入"标签
- 上传Excel文件
- 选择工作表
- 配置输入列数量和对应列
- 点击"从Excel加载输入"

#### 方式二：手动添加
- 点击"手动添加"标签
- 在文本框中输入数据，每行一条
- 点击"添加输入"

### 2. 配置模板
- 选择预置模板或新建模板
- 输入模板名称
- 选择AI模型和温度参数
- 配置系统提示和用户模板
- 定义输出字段（JSON格式）
- 点击"保存模板"

### 3. 创建任务
- 为每个模板指定要处理的输入数据范围
- 点击"添加到请求列表"
- 预览请求列表，确认无误后点击"提交全部请求"

### 4. 查看结果
- 在"任务与结果"标签页查看任务状态
- 点击结果列查看详细结果
- 点击"导出为Excel"导出结果

## 功能亮点

- **灵活的输入方式**：支持Excel导入和手动添加，适应不同数据来源
- **可复用的模板系统**：创建的模板可重复使用，提高工作效率
- **直观的任务管理**：可视化任务状态，便于跟踪和管理
- **结构化结果输出**：支持自定义JSON输出格式，便于后续数据处理
- **高效的批量处理**：能够快速处理小批量专利数据

## 注意事项

1. Excel文件大小建议不超过10MB，避免影响处理速度
2. 建议将相关数据放在同一工作表中，便于系统识别
3. 模板配置时，系统提示和用户模板应清晰明确，指导AI生成准确结果
4. 输出字段定义应符合JSON格式规范
5. 提交请求前，建议先预览请求列表，确认无误后再提交
6. 任务处理时间取决于数据量和模型性能，建议耐心等待
7. 结果导出时，建议及时保存，避免数据丢失
