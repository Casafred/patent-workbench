## 技术说明

### 按钮尺寸优化原则
- 主要操作按钮: `padding: 8-12px 16-24px`
- 次要操作按钮: `padding: 6-8px 12-16px`
- 小型辅助按钮: `padding: 4-6px 10-12px`
- 微型按钮: `padding: 2-4px 6-8px`

### Hover状态设计原则
- 保持按钮的主色调不变
- 使用 `filter: brightness()` 或 `transform` 提供视觉反馈
- 避免颜色突变（如从绿色变成白色）
- 确保文字始终清晰可读
