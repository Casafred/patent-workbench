功能六：弹窗关闭方式优化 - 左上角关闭按钮

- 移除点击遮罩层关闭弹窗的功能

- 关闭按钮悬停时变为红色，提供清晰的交互反馈
- 调整标题布局，为关闭按钮留出空间

- 操作更明确，避免误关闭弹窗

- 适合需要仔细查看专利详情的场景
