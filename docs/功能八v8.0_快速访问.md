# 功能八 v8.0 - 快速访问指南

## 🚀 快速开始

### 1. 打开功能八
- 在主页面点击"功能八：专利附图标记"标签

### 2. 上传附图
- 点击"选择文件"上传专利附图（支持多张）
- 点击"开始 OCR 识别"

### 3. 打开多图查看器
- 识别完成后，点击绿色的"打开多图查看器"按钮
- **注意**：按钮已去掉 EMOJI，只有 SVG 图标

## 🎨 多图查看器功能

### 左侧：图片显示区
- **左右切换**：鼠标移到边缘显示箭头，点击切换
- **键盘导航**：← → 方向键切换图片
- **拖动查看**：按住鼠标拖动查看大图
- **滚轮缩放**：滚动鼠标滚轮放大/缩小
- **双击添加**：双击位置添加新标注
- **单击选择**：单击标注选中（绿色高亮）
- **Ctrl+单击**：多选标注
- **双击编辑**：双击标注文字编辑名称
- **右键删除**：右键标注删除

### 右侧：功能控制栏

#### 📊 图片信息
- 显示当前图片序号和总数

#### 🔤 字体大小
- **选中-/+**：调整选中标注字体
- **全部-/+**：调整所有标注字体

#### 🔄 图片旋转
- **↺ 逆时针**：逆时针旋转 90°
- **↻ 顺时针**：顺时针旋转 90°
- 标注点会跟随旋转，文字保持可读

#### 🔍 缩放
- **-**：缩小
- **重置**：恢复 100%
- **+**：放大

#### ✅ 选择控制
- **全选**：选中所有标注
- **取消选择**：取消所有选中

#### 🎨 标注颜色
- 8 种预设颜色自动循环分配
- 点击颜色改变选中标注的颜色
- 橙色、绿色、蓝色、紫色、红色、青色、黄色、粉色
- 选中时保持原有颜色，边框加粗

#### 📸 高清截图 ⭐ 新功能
- **橙色按钮**：点击导出高清 PNG 图片
- 包含所有标注、旋转状态、颜色
- 文件名：`{图片名称}_标注_{时间戳}.png`

#### 🔧 调试面板
- 显示 OCR 识别结果
- 显示说明书提取结果
- 对比分析

#### ✕ 关闭
- 关闭多图查看器

## 💡 使用技巧

### 快速标注
1. 双击图片位置添加标注
2. 输入序号和说明
3. 标注会自动偏移，避免遮挡原图

### 批量调整
1. 点击"全选"按钮
2. 使用"全部+/-"调整字体
3. 选择颜色统一标注颜色

### 精确定位
1. 单击标注选中（绿色高亮）
2. 拖动标注到精确位置
3. 标注框不会超出画布边界

### 导出分享
1. 调整好所有标注
2. 点击"📸 高清截图"
3. 自动下载 PNG 图片
4. 可用于报告、演示、分享

## 🎯 最佳实践

### 标注流程
1. **OCR 识别** → 自动识别附图标记
2. **检查调整** → 检查识别结果，手动添加遗漏
3. **美化排版** → 调整字体、颜色、位置
4. **导出保存** → 高清截图保存结果

### 颜色使用建议
- **橙色**：主要部件
- **绿色**：次要部件
- **蓝色**：连接部件
- **紫色**：特殊部件
- **红色**：重点标注
- **其他**：根据需要自定义

## 📱 快捷键

- **← →**：切换图片
- **Esc**：关闭查看器
- **Ctrl+单击**：多选标注
- **双击**：添加/编辑标注
- **右键**：删除标注
