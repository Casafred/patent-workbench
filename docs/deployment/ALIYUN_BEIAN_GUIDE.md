## ⚠️ 备案注意事项

### 必须做的调整

#### 1. 去除 AI 聊天功能的显著展示

**问题：** 备案时，AI 对话功能可能被认为是"智能客服"或"聊天室"，需要额外审批。

**解决方案：**
```
方案 A：隐藏功能（推荐）
- 备案期间暂时隐藏"功能一：即时对话"
- 备案通过后再开放
- 或者改名为"数据查询助手"

方案 B：弱化描述
- 不在首页突出显示
- 描述为"辅助查询功能"而非"AI聊天"
```

#### 2. 添加必要的法律声明

在网站底部添加：

```html
<!-- 在 frontend/index.html 底部添加 -->
<footer class="site-footer">
    <div class="footer-content">
        <p>© 2025 [你的公司名/个人名] 版权所有</p>
        <p>
            <a href="/terms.html">服务条款</a> | 
            <a href="/privacy.html">隐私政策</a> | 
            <a href="/disclaimer.html">免责声明</a>
        </p>
        <p>ICP备案号：<a href="https://beian.miit.gov.cn/" target="_blank">京ICP备XXXXXXXX号</a></p>
    </div>
</footer>
```

#### 3. 创建必要的法律页面

需要创建以下页面：

**a. 服务条款 (terms.html)**
```
- 服务范围说明
- 用户责任
- 知识产权声明
- 免责条款
```

**b. 隐私政策 (privacy.html)**
```
- 数据收集说明（强调不存储用户数据）
- 数据使用说明
- 数据安全措施
- Cookie 使用说明
```

**c. 免责声明 (disclaimer.html)**
```
- 本工具仅供数据分析使用
- 不提供法律咨询服务
- 不对分析结果的准确性负责
- 用户需自行承担使用风险
```

#### 4. 添加用户协议确认

在登录页面添加：

```html
<label>
    <input type="checkbox" id="agree_terms" required>
    我已阅读并同意 <a href="/terms.html" target="_blank">服务条款</a> 
    和 <a href="/privacy.html" target="_blank">隐私政策</a>
</label>
```

## 📋 备案流程（详细步骤）

### 第一步：准备域名（1天）

1. **购买域名**
   - 推荐：阿里云万网
   - 选择 `.com`、`.cn`、`.net` 等常见后缀
   - 费用：¥50-100/年

2. **域名实名认证**
   - 上传身份证/营业执照
   - 等待审核（1-3小时）

3. **获取域名证书**
   - 在域名控制台下载
   - 备案时需要上传

### 第二步：购买服务器（已完成）

- ✅ 你已经准备购买阿里云 ECS
- ✅ 服务器必须在中国大陆（已满足）
- ✅ 购买时长至少 3 个月（建议包年）

### 第三步：提交备案申请（1天）

1. **登录阿里云备案系统**
   - 访问：https://beian.aliyun.com/
   - 使用购买 ECS 的账号登录

2. **填写主体信息**
   - 企业：营业执照信息
   - 个人：身份证信息

3. **填写网站信息**
   - 网站名称：参考上面的建议
   - 网站内容：工具软件
   - 网站描述：参考上面的模板

4. **上传材料**
   - 身份证/营业执照
   - 域名证书
   - 网站负责人信息

5. **拍摄幕布照片**
   - 使用阿里云 APP 拍摄
   - 按照要求拍摄幕布照片

### 第四步：等待初审（1-2天）

- 阿里云会先审核材料
- 如有问题会电话/短信通知
- 及时补充材料

### 第五步：管局审核（10-20天）

- 初审通过后提交到省通信管理局
- 管局会进行最终审核
- 可能会电话核实信息

### 第六步：备案通过

- 收到备案号（如：京ICP备12345678号）
- 在网站底部添加备案号
- 链接到 https://beian.miit.gov.cn/

## 🚨 常见问题

### Q1: 个人备案可以做企业工具吗？

**A:** 可以，但要注意：
- 网站名称不能含"公司"、"企业"
- 描述为"个人开发的工具"
- 说明"免费提供给用户使用"

### Q2: 备案期间网站能访问吗？

**A:** 不能。
- 备案期间域名不能解析
- 可以用 IP 访问测试
- 备案通过后才能用域名访问

### Q3: 备案被拒怎么办？

**A:** 根据拒绝原因调整：
- 网站名称不合规 → 修改名称
- 网站内容不明确 → 补充说明
- 材料不清晰 → 重新上传
- 一般可以重新提交

### Q4: 备案后可以改网站内容吗？

**A:** 可以，但注意：
- 不要改变网站性质
- 不要添加需要前置审批的内容
- 重大变更需要变更备案

### Q5: 不备案可以吗？

**A:** 不可以。
- 使用国内服务器必须备案
- 不备案会被关停
- 建议：先用 Render，备案通过后迁移
