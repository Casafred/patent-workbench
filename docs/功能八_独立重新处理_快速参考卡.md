# 功能八：独立重新处理 - 快速参考卡 🚀

## 🎯 两种模式

### 📝 模式1：重新处理说明书
**场景**: 图片已OCR识别，只需更新说明书  
**操作**: 点击"📝 重新处理说明书"按钮  
**速度**: 2-5秒（节省70%时间）  
**API**: `/api/drawing-marker/reprocess-specification`

### 🖼️ 模式2：重新识别图片
**场景**: 说明书已解析，只需重新上传图片  
**操作**: 点击"🖼️ 重新识别图片标号"按钮  
**速度**: 5-10秒（节省30%时间）  
**API**: `/api/drawing-marker/reprocess-drawings`
