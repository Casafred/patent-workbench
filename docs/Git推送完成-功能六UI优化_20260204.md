## 💡 使用提示

### 左侧导航使用
1. 点击任意导航项，页面平滑滚动到对应区域
2. 滚动页面时，导航自动高亮当前区域
3. 悬停导航项时，背景变绿并向右平移

### 弹窗Header使用
1. 标题区域显示专利号和标题
2. 查询耗时显示在标题下方
3. 右侧按钮从左到右：
   - 上一条/下一条：切换专利
   - 新标签页：在新标签页打开
   - 问一问：打开AI对话
   - 关闭：关闭弹窗
