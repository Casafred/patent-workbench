## 🐛 常见问题排查

### 问题1：弹窗点击外部无法关闭
**可能原因：**
- 浏览器缓存未清除
- JavaScript未正确加载

**解决方法：**
```
1. 按Ctrl+Shift+R强制刷新页面
2. 打开开发者工具(F12)查看Console是否有错误
3. 检查Network标签，确认js/main.js已加载
```

### 问题2：新标签页没有摘要
**可能原因：**
- 专利数据中没有摘要字段
- JavaScript未正确加载

**解决方法：**
```
1. 尝试其他专利号
2. 打开开发者工具查看Console错误
3. 检查js/patentDetailNewTab.js是否加载
```

### 问题3：复制功能不工作
**可能原因：**
- 浏览器不支持Clipboard API
- 需要HTTPS环境

**解决方法：**
```
1. 确认使用的是现代浏览器（Chrome/Edge/Firefox最新版）
2. 确认网站使用HTTPS协议
3. 检查浏览器权限设置
```

### 问题4：折叠功能不工作
**可能原因：**
- CSS样式未加载
- JavaScript事件未绑定

**解决方法：**
```
1. 检查开发者工具Console是否有错误
2. 检查Elements标签，确认section有collapsible-section类
3. 检查section是否有collapsed类
```

## 💡 测试技巧

1. **使用开发者工具**
   - 按F12打开开发者工具
   - Console标签查看错误信息
   - Network标签查看资源加载
   - Elements标签查看DOM结构

2. **测试不同专利号**
   - 有引用专利的：US10000000B2
   - 有被引用的：US9000000B2
   - 有相似文档的：CN104154208B

3. **测试不同浏览器**
   - Chrome（推荐）
   - Edge
   - Firefox
   - Safari

4. **测试不同屏幕尺寸**
   - 桌面端（1920x1080）
   - 笔记本（1366x768）
   - 平板（768x1024）
