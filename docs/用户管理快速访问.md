# 用户管理快速访问

## 🚀 方式一：独立HTML页面（最简单，推荐）

### 使用步骤

1. **直接双击打开**：
   - 找到项目根目录下的 `用户管理.html` 文件
   - 双击打开即可使用
   - **无需启动任何服务器**

2. **导入现有 users.json**：
   - 点击或拖拽 `backend/user_management/users.json` 到导入区域
   - 系统会自动加载并显示所有现有用户
   - 支持拖拽上传，非常方便

3. **管理用户**：
   - **查看用户**：导入后自动显示所有用户、注册时间和备注信息
   - **添加用户**：在"添加新用户"区域输入用户名、密码和备注（可选）
   - **删除用户**：在用户列表中点击"删除"按钮

4. **保存更改**：
   - 点击"💾 下载 users.json"按钮
   - 浏览器会下载更新后的 `users.json` 文件
   - 将下载的文件替换到 `backend/user_management/users.json`
   - 或者点击"📋 复制JSON到剪贴板"直接复制内容

### 特点

- ✅ 无需后端服务器
- ✅ 界面友好，操作简单
- ✅ **支持导入现有 users.json 文件**
- ✅ **可以查看所有现有用户**
- ✅ 自动密码加密（PBKDF2-SHA256）
- ✅ 支持拖拽上传文件
- ✅ 实时预览用户列表
- ✅ 支持删除用户
- ✅ 一键复制到剪贴板

## 🌐 方式二：通过浏览器访问（需要启动服务器）

1. 启动应用：
   ```bash
   python run_app.py
   ```

2. 在浏览器中访问：
   ```
   http://localhost:5000/user-management
   ```

3. 在界面中可以：
   - 查看所有用户
   - 添加新用户
   - 删除用户
   - 实时保存到服务器

## 📝 示例

### 使用独立HTML页面
1. 双击 `用户管理.html`
2. 拖拽 `backend/user_management/users.json` 到导入区域
3. 查看现有用户列表
4. 添加新用户：输入用户名 `admin`，密码 `admin123`
5. 点击"添加用户"
6. 点击"下载 users.json"
7. 将下载的文件替换到 `backend/user_management/users.json`

### 使用在线管理页面
1. 启动应用：`python run_app.py`
2. 访问：`http://localhost:5000/user-management`
3. 在界面中添加/删除用户
4. 实时保存到服务器

---

## ⚠️ 注意事项

1. 用户信息存储在 `backend/user_management/users.json`
2. 密码会自动加密存储（PBKDF2-SHA256算法）
3. 使用方式二或三修改后需要重启应用才能生效
4. 使用方式一需要手动替换 `users.json` 文件
5. 建议定期备份 `users.json` 文件
