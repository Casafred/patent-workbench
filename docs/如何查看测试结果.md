# 如何查看标记布局算法测试结果 🎯

## 🚀 快速开始

### 方法1：运行Python模拟测试（推荐）

```bash
python test_label_layout_simulation.py
```

**输出内容**:
- 14个测试场景的详细对比
- 旧算法 vs 新算法的数据
- 统计分析和改进百分比
- 测试结论

**预期结果**:
```
✅ 新算法更接近标记点的场景: 10/14 (71.4%)
📏 平均到标记点距离:
   旧算法: 63.6px
   新算法: 37.1px
   改进: 26.4px (41.6%)
```

- 📏 距离信息显示

**统计指标**:
- 平均到标记点距离（旧/新）
- 距离改进百分比
- 更优场景占比
