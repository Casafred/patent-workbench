## 🎯 高清截图功能

### 功能特点
- 导出包含所有标注的高清 PNG 图片
- 保留当前旋转状态
- 包含所有标注点、连接线、标注文字
- 保持选中状态的高亮效果（绿色）
- 保持每个标注的自定义颜色

### 文件命名
格式：`{图片名称}_标注_{时间戳}.png`

示例：`图1_标注_2026-02-01T12-30-45.png`
