## 🎯 设计理念

### 为什么选择绿色系？

1. **视觉一致性**
   - 专利详情页使用绿色主题
   - 字段选择器改为绿色系后，整体风格统一

2. **色彩心理学**
   - 绿色代表：自然、成长、和谐、稳定
   - 适合专利分析这种需要专注和准确的场景

3. **可读性**
   - 绿色系对眼睛更友好
   - 长时间使用不易疲劳

4. **品牌识别**
   - 绿色系可以成为应用的特色
   - 与其他专利工具形成差异化

## 📐 颜色使用规范

### 主色调（Green 600 - #43a047）
- 用途：图标、按钮、复选框、边框强调
- 特点：鲜明、醒目、易识别

### 深色（Green 700 - #2e7d32）
- 用途：标题、重要文字、按钮文字
- 特点：稳重、专业、易阅读

### 更深色（Green 800 - #1b5e20）
- 用途：选中状态文字、强调文字
- 特点：对比度高、突出重点

### 淡色（Green 100 - #c8e6c9）
- 用途：边框、选中背景、分隔线
- 特点：柔和、不刺眼、舒适

### 极淡色（Green 50 - #e8f5e9）
- 用途：悬停背景、次要背景
- 特点：极其柔和、几乎不可见

### 自定义淡绿（#f1f8f4）
- 用途：字段选项默认背景
- 特点：介于白色和淡绿之间，非常舒适

## 🌈 颜色应用示例

### CSS 代码片段

```css
/* 主色调 */
.field-selector-title svg {
    color: #43a047;  /* Green 600 */
}

/* 深色文字 */
.field-selector-title {
    color: #2e7d32;  /* Green 700 */
}

/* 淡色边框 */
.field-selector-container {
    border: 1px solid #c8e6c9;  /* Green 100 */
}

/* 选中状态 */
.field-option.checked {
    background: #c8e6c9;  /* Green 100 */
    border-color: #43a047;  /* Green 600 */
}

.field-option.checked .field-option-label {
    color: #1b5e20;  /* Green 800 */
}

/* 悬停效果 */
.field-option:hover {
    background: #e8f5e9;  /* Green 50 */
    border-color: #43a047;  /* Green 600 */
}

/* 复选框 */
.field-option input[type="checkbox"] {
    accent-color: #43a047;  /* Green 600 */
}
```
