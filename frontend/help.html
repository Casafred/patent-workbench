入门指南

欢迎使用 ALFRED X IP 专利分析智能工作台！本手册将引导您快速上手，充分利用本平台的强大功能。

1. 登录系统

您需要使用管理员分配的用户名和密码进行登录。为保障安全，系统会记录并限制每个账户的登录IP地址数量（默认为5个）。如果您在新的设备或网络环境下登录，最早的登录记录可能会被自动移除。

[此处插入动图：展示用户输入用户名、密码并成功登录的过程]

2. 界面概览

登录后，您将看到主工作界面，主要分为三个区域：

  • 顶部导航栏： 显示Logo、版本号，以及右上角的"帮助"和"API Key设置"按钮。
  • 功能页签区： 切换八个核心功能模块。
  • 主内容区： 显示当前选中功能模块的具体操作界面。
[此处插入图片：用箭头和文字标注主界面的各个区域]

3. API Key 设置（最重要的一步！）

本平台的所有AI功能都依赖于智谱AI提供的API服务。因此，在使用任何功能前，您必须配置自己的智谱AI API Key。

  1. 点击界面右上角的 ⚙️ 图标，打开API Key设置面板。
  2. 将您从智谱AI开放平台获取的API Key粘贴到输入框中。
  3. 点击"保存"按钮。看到"已保存!"提示即表示配置成功。
注意：API Key是您的个人凭证，请妥善保管，切勿泄露。所有通过本平台发起的AI调用都将消耗您API Key对应的账户额度。
[此处插入动图：展示点击设置图标、粘贴API Key并保存的全过程]

4. 登出系统

在主界面右上角，您可以看到当前登录的用户名和一个"登出"按钮。点击"登出"将安全退出系统，并清除您的会话信息。

功能一：即时对话

本功能提供了一个强大的、类似ChatGPT的交互界面，专为专利分析场景优化，支持多轮对话、角色扮演、文件上传解析和历史记录管理。

核心交互

在底部的输入框中输入您的问题，按回车或点击"发送"按钮即可与AI进行实时对话。AI的回复将以流式方式逐字显示。

对话管理

  • 历史对话列表： 左侧面板会显示您所有的历史对话，按时间倒序排列。
  • 新建对话： 点击列表顶部的"+"号或输入框左侧的"+"号，可以随时开启一个全新的对话。
  • 切换对话： 在历史列表中点击任意对话，即可加载并继续该对话。
  • 重命名与删除： 将鼠标悬停在历史对话项上，会出现编辑和删除图标，方便您管理对话。您也可以直接点击主聊天窗口的标题进行修改。
[此处插入动图：展示新建、切换、重命名和删除对话的操作]

对话参数设置

点击右上角的"对话参数设置"按钮，可以打开配置面板：

  • 模型选择： 选择不同的AI模型以应对不同复杂度的任务。
  • 温度（Temperature）： 控制回复的创造性。值越高（如0.9）回复越随机，值越低（如0.1）回复越确定。
  • 上下文条数： 决定每次请求时携带多少轮历史对话作为上下文，影响对话的连贯性和费用。
  • 角色设置： 这是本功能的核心特色之一。您可以选择预设的"资深专利分析师"、"专业技术翻译"等角色，或自定义新角色。

角色（Persona）管理

在"对话参数设置"中，您可以：

  1. 选择角色： 从下拉菜单中选择一个角色，当前对话将立刻采用该角色的系统提示。
  2. 新增角色： 点击"新增角色"，可以创建一个自定义角色，定义其名称、系统提示（System Prompt）和用户模板。
  3. 保存与删除： 自定义角色可以随时编辑、保存或删除。
用户模板提示： 在用户模板中使用 {{INPUT}} 作为占位符，系统会自动将您在输入框中输入的内容替换到该位置。例如，模板为 请将以下内容翻译成英文：{{INPUT}}，您只需在输入框输入中文即可。
[此处插入动图：演示如何新增一个自定义角色，并应用到当前对话中]

文件处理与管理

即时对话支持上传文件（如PDF, Word, TXT, Excel, 图片等）进行内容分析。

  • 上传文件： 点击输入框左侧的回形针📎图标，选择文件进行上传。系统会自动提取文件内容。
  • 解析服务选择： 上传文件后，可选择不同的解析服务：
    • Lite（免费）： 满足日常查询需求，支持常见格式，输出纯文本
    • Expert（0.03元/次）： 适合专业PDF解析，输出Markdown格式并保留图片
    • Prime（0.05元/次）： 支持最多格式，保留完整文档结构
  • 文件管理： 点击标题栏的"文件管理"按钮，可以查看、删除您已上传的所有文件，或将文件内容直接插入到输入框中。
  • 移除文件： 文件附加后，输入框上方会显示文件名，点击旁边的"×"可将其从当前对话中移除。
[此处插入动图：演示点击回形针上传文件，AI根据文件内容进行回答的过程]

联网搜索

点击输入框左侧的搜索🔍按钮，可以开启联网搜索功能。系统会使用智谱网络搜索API，在回答问题时参考最新的网络信息。

提示： 联网搜索功能需要消耗额外的API额度，建议在需要获取最新信息时使用。

消息管理与导出

  • 管理消息： 点击"管理消息"按钮，可以进入多选模式，批量删除不需要的对话记录。
  • 导出： 点击"导出"按钮，可以将当前完整的对话记录导出为TXT、PNG图片或PDF文件，方便存档和分享。

功能二：小批量异步

此功能适用于处理中等数量（几十到几百条）的独立任务，例如批量翻译、批量总结等。它通过异步API提交任务，无需等待即可继续操作，并在后台轮询结果。

工作流程

整个过程遵循顶部的四步进式导航：

  1. 输入内容 →
  2. 配置模板 →
  3. 创建任务 →
  4. 任务与结果

Step 1: 输入内容

您有两种方式添加需要处理的源数据：

  • Excel导入： 上传Excel文件，选择工作表(Sheet)，然后指定包含源数据的一列或多列。
  • 手动添加： 在文本框中直接输入，每行代表一条独立的输入数据。
[此处插入动图：演示如何从一个Excel文件的特定列导入数据]

Step 2: 配置模板

模板定义了如何处理每一条输入数据。一个模板包含：

  • 模板名称： 为您的处理方式命名，方便复用。
  • 模型与温度： 同即时对话。
  • 系统提示： 定义AI的角色和总体指令。
  • 用户提示模板： 核心部分，必须包含 {{INPUT}} 占位符。系统会将每条输入数据替换到这里。
  • 输出格式(JSON)： 这是一个强大的功能。您可以定义希望AI输出的JSON字段名和描述，系统会自动生成严格的JSON格式要求附加到提示中，确保AI返回结构化数据。
[此处插入动图：展示如何创建一个"技术总结"模板，并定义"技术方案"、"技术问题"两个JSON输出字段]

Step 3: 创建任务

此步骤将"输入"和"模板"进行配对。您可以为每个模板指定要处理的输入数据序号范围（如 1-50, 60, 70-80），然后将其添加到下方的"待提交请求列表"。

Step 4: 任务与结果

  1. 提交任务： 点击"提交全部请求"，系统会将所有请求发送到异步处理队列。
  2. 查看结果： 提交后，下方的表格会实时更新每个请求的状态（处理中、成功、失败）。
  3. 恢复任务： 如果您关闭了浏览器，可以点击"恢复上次任务"按钮，从浏览器缓存中加载未完成的任务并继续轮询。
  4. 导出Excel： 任务完成后，点击"导出为Excel"按钮，可以将所有输入、状态和AI返回的结果合并到一个Excel文件中。

功能三：大批量处理

专为大规模数据处理（数千至数万条）设计，基于智谱AI的官方Batch API。此功能完全在后端执行，即使关闭浏览器，任务也会继续运行，适合耗时较长的分析任务。

工作流程

同样采用三步进式导航：

  1. 生成请求文件 →
  2. 执行批处理 →
  3. 解析报告

Step 1: 生成请求文件

此步骤的目的是根据您的源数据和处理要求，在浏览器内存中生成一个符合Batch API格式的JSONL文件。

  1. 上传源数据： 上传包含大量专利数据的Excel文件。
  2. 配置分析列和模板： 与"小批量异步"类似，配置需要分析的列和处理模板（包含System Prompt、User Prompt和JSON输出格式）。
  3. 生成JSONL： 点击"生成JSONL文件"按钮，系统会为Excel中的每一行生成一条API请求，并预览前几条。此时，文件已在内存中准备就绪。
[此处插入动图：演示从上传Excel到配置模板，最终生成JSONL预览的过程]

Step 2: 执行批处理

这是与Batch API交互的核心环节。

  1. 上传请求文件： 点击按钮，将上一步在内存中生成的JSONL文件上传到智谱AI。
  2. 创建Batch任务： 上传成功后，点击此按钮创建一个批处理任务。请务必复制并保管好返回的Batch ID，这是您找回任务的唯一凭证。
  3. 检查状态： 创建任务后，系统会自动启动定时检查（每分钟一次）。您也可以手动点击检查。
  4. 获取结果内容： 当任务状态变为"completed"时，此按钮将可用。点击后，系统会下载结果文件并加载到内存中，然后自动跳转到下一步。
断点续查： 如果流程中断，您可以在"断点续查"区域输入您之前保存的Batch ID，然后点击"恢复查询状态"来继续监控任务。

Step 3: 解析报告

此步骤将AI返回的结果与您的原始数据合并，生成最终的分析报告。

  1. 准备数据源： 上传您在第一步使用的原始Excel文件。同时，系统会自动检测并加载上一步从Batch任务下载的结果数据。
  2. 生成报告： 点击"解析并生成最终报告"，系统会将结果中的JSON数据解析出来，并作为新的列附加到原始数据的每一行后面。
  3. 下载报告： 生成成功后，点击按钮即可下载包含所有原始数据和AI分析结果的最终Excel文件。

功能四：本地专利库管理

这是一个纯粹的、离线的Excel数据处理工具，专注于专利建库过程中常见的两个痛点：同族展开和新旧库合并。此功能不消耗任何API额度。

工作流程

系统采用进度步进器引导您完成整个流程：

  1. 上传与展开 →
  2. 合并新旧库 →
  3. 下载最终结果

Step 1: 上传与展开

此功能用于处理包含同族专利号的字段。

  1. 上传文件： 上传您的原始专利库Excel文件。
  2. 配置： 指定包含同族专利号的列名（如 INPADOC 同族专利成员）和分隔符（如 |）。
  3. 展开： 点击"展开同族专利号"按钮。系统会读取指定列的所有内容，按分隔符拆分，然后生成一个去重后的、包含所有独立专利号的列表。
  4. 导出： 您可以一键复制该列表，或将其下载为.txt文件，用于后续的数据库检索。
[此处插入动图：演示上传文件，指定列名和分隔符，然后生成去重后的专利号列表]

Step 2: 合并新旧库

当您从商业数据库下载了最新的数据，需要将其与您自己维护的、包含额外标注（如技术分类、评级等）的旧库合并时，此功能非常有用。

  1. 加载文件： 分别上传您的旧库（原始库）和从数据库下载的新库。
  2. 配置匹配列： 为两个文件分别指定一个用于匹配的"主键"列，通常是"公开号"。
  3. 选择补充列： 从旧库中勾选您希望补充到新库的列（例如您手动添加的"技术评级"、"备注"等字段）。
  4. 开始合并： 点击按钮。系统会以新库为基础，遍历每一行数据，通过匹配列在旧库中查找对应的行。如果找到，就将您勾选的补充列的数据填充到新库的对应行中。
合并逻辑： 新库的数据会被完整保留。如果新库中本身就存在与旧库补充列同名的列，则新库的数据优先，不会被覆盖。

Step 3: 下载最终结果

合并完成后，系统会自动跳转到此页面。

  • 预览： 您可以预览合并后数据的前10行。表格的列宽可以手动拖动调整。
  • 下载： 点击按钮，即可下载包含新库所有数据以及从旧库补充的数据的最终Excel文件。

功能五：权利要求对比

此功能用于智能对比不同专利的权利要求，自动识别语言、按需翻译，提供多种可视化展示方式，帮助您快速识别它们之间的相似性和差异。

核心技术优势

  • 多模型支持： 提供多种模型选择（从配置文件动态加载），满足不同场景需求。
  • 灵活对比数量： 支持2-10个权利要求的同时对比，适应不同的对比分析需求。
  • 智能语言处理： 自动识别权利要求语言，支持中英文混合文本，按需提供翻译功能。
  • 多种可视化方式： 提供卡片视图、并排对比、矩阵视图三种展示方式，直观呈现对比结果。
  • 耦合对比分析： 支持选择多个权利要求进行耦合对比，分析它们之间的关联关系。
  • 实时统计分析： 自动计算并显示相同特征、差异特征和平均相似度，提供量化的对比结果。

工作流程

  1. 输入权利要求： 在输入框中输入专利的权利要求内容，点击"添加"按钮可增加更多权利要求（最多10个）。
  2. 选择对比模型： 根据需求选择合适的对比模型。
  3. 执行智能对比： 点击"开始全部对比分析"按钮，系统自动识别语言、计算相似度并生成对比结果。
  4. 耦合对比分析（可选）： 在"耦合对比分析"区域选择多个权利要求，点击"开始耦合分析"进行深度关联分析。
  5. 切换视图： 根据需要切换不同的可视化展示方式（卡片视图/并排对比/矩阵视图）。
  6. 导出报告： 将对比结果导出为Excel或PDF文件，方便进一步分析和分享。

可视化视图说明

  • 卡片视图： 以卡片形式展示每个权利要求的核心特征和对比结果。
  • 并排对比： 将多个权利要求并排显示，便于逐条对比差异。
  • 矩阵视图： 以矩阵形式展示权利要求之间的相似度关系，适合多维度分析。

使用技巧

  • 对于简单对比，建议使用轻量级模型，速度更快。
  • 对于复杂技术领域的深度对比，建议使用高质量模型，结果更准确。
  • 使用"并排对比"视图可以直观比较不同权利要求的差异。
  • 使用"矩阵视图"可以快速了解多个权利要求之间的相似度关系。
  • 通过"切换为译文"功能，可以查看不同语言权利要求的翻译结果。
  • 耦合对比分析适合分析权利要求之间的引用关系和依赖关系。

应用场景

  • 专利侵权分析：对比涉嫌侵权专利与现有专利的权利要求。
  • 专利无效分析：对比目标专利与现有技术的权利要求差异。
  • 技术演进分析：对比同一技术领域不同时期专利的权利要求变化。
  • 专利布局分析：对比不同竞争对手的专利权利要求策略。
  • 权利要求引用分析：通过耦合对比分析权利要求之间的引用关系。

功能六：批量专利解读

此功能用于批量查询专利详情并提供AI解读功能，支持专利号列表输入、自动爬取专利信息、智能解读和结果导出，最多支持50个专利号。

核心技术优势

  • 高效批量处理： 支持一次输入最多50个专利号，批量查询和解读，大幅提高工作效率。
  • 智能专利爬取： 自动从专利数据库获取专利详情，支持说明书内容爬取，为AI解读提供完整数据。
  • 灵活字段选择器： 提供详细的字段选择功能，可选择需要爬取的专利信息字段，平衡数据完整性和处理时间。
  • 解读模板管理： 支持多种解读模板（默认/技术分析/商业价值/法律分析），可自定义模板内容。
  • 实时数量统计： 自动统计输入的专利号数量，防止超过限制。
  • 一键批量解读： 支持对所有查询到的专利进行一键解读，操作便捷。
  • 结构化结果输出： 解读结果以结构化形式展示，便于分析和比较。
  • Excel导出功能： 支持将解读结果导出为Excel文件，方便进一步分析和分享。

字段选择器说明

系统提供详细的字段选择功能，分为以下几类：

  • 基础字段（必选）： 专利号、标题、摘要、发明人、申请人、申请日期、公开日期、权利要求
  • 分类与领域： CPC分类、技术领域、优先权日期
  • 同族信息： 同族ID、同族申请、国家状态
  • 引用信息： 引用专利、被引用专利
  • 法律与事件： 事件时间轴、法律事件、相似文档
  • 附加内容： 说明书、附图、外部链接
提示： 部分字段（如说明书、同族申请）标记为"耗时"，选择这些字段会增加爬取时间。建议根据实际需求选择字段。

工作流程

  1. 输入专利号列表： 在文本框中输入专利号，用换行符或空格分隔，最多50个。
  2. 选择解读模板： 从下拉菜单中选择合适的解读模板，或点击"管理"自定义模板。
  3. 配置爬取字段： 点击"选择爬取字段"按钮，勾选需要获取的专利信息字段。
  4. 批量查询专利： 点击"批量查询专利"按钮，系统自动爬取专利详情。
  5. 执行AI解读： 点击"批量解读"按钮，对所有查询到的专利进行AI解读。
  6. 查看解读结果： 在结果列表中查看每个专利的解读内容，点击可查看详情。
  7. 导出Excel报告： 将所有专利的解读结果导出为Excel文件。

使用技巧

  • 专利号输入支持多种分隔方式，包括换行符、空格、逗号等。
  • 对于需要深度分析的专利，建议选择"说明书"和"附图"字段，可获得更全面的解读结果。
  • 对于仅需要基本信息的快速分析，建议只选择基础字段，处理速度更快。
  • 解读模板可以自定义，根据分析需求调整解读角度（技术/商业/法律）。
  • 解读结果中包含专利的技术领域、技术问题、技术方案和有益效果等关键信息。
  • 导出的Excel文件包含所有专利的解读数据，方便进行进一步的统计分析。

应用场景

  • 技术调研：快速了解多个相关专利的技术内容和创新点。
  • 竞争对手分析：批量解读竞争对手的专利，了解其技术布局和发展方向。
  • 专利挖掘：从大量专利中筛选出有价值的技术信息。
  • 专利预警：定期批量解读相关领域的新专利，及时了解技术动态。
  • 专利导航：为企业的技术研发和专利布局提供参考。

功能七：权利要求处理

此功能用于处理专利权利要求，支持Excel批量分析和文本输入分析两种模式，能够自动解析、分类并提取权利要求信息，生成可视化的引用关系图。

核心技术优势

  • 双模式支持： 提供Excel批量分析和文本输入分析两种模式，适应不同的使用场景。
  • 智能文本解析： 自动识别权利要求的编号、类型（独立/从属）、内容和引用关系。
  • 多语言支持： 支持中英文混合文本，自动识别权利要求语言。
  • 置信度评分： 为每条权利要求的解析结果提供置信度评分，帮助用户评估解析准确性。
  • 多种可视化方式： 提供树状图、网络图、径向图三种引用关系可视化方式，直观展示权利要求结构。
  • 交互式可视化控制： 支持缩放、拖拽、散开程度调节、高清截图等交互功能。
  • 公开号与独权合并： 提供公开号与独权合并视窗，方便查看每个公开号及其对应的独立权利要求。
  • 专利号查询功能： 支持根据专利号搜索和可视化特定专利的权利要求引用关系。
  • 多种导出格式： 支持将处理结果导出为Excel、JSON格式，方便进一步分析和应用。
  • 实时进度显示： 处理过程中显示实时进度条和百分比，让用户了解处理状态。

工作流程（Excel批量分析模式）

  1. 上传Excel文件： 选择包含权利要求的Excel文件。
  2. 选择工作表和列： 从下拉菜单中选择包含权利要求的工作表和列。
  3. 开始处理： 点击"开始处理"按钮，系统自动解析权利要求信息。
  4. 查看处理结果： 在表格中查看结构化的权利要求信息，包括类型、语言、引用关系和置信度。
  5. 查看公开号与独权合并： 在合并视窗中查看每个公开号的独立权利要求内容。
  6. 搜索专利号： 输入专利号片段搜索，定位到特定专利。
  7. 生成可视化图表： 选择特定专利，生成权利要求引用关系图。
  8. 导出结果： 将处理结果导出为Excel或JSON文件。

工作流程（文本输入分析模式）

  1. 输入权利要求文本： 在文本框中输入权利要求内容，或使用"加载示例"查看示例格式。
  2. 开始处理： 点击"开始分析"按钮，系统自动解析权利要求信息。
  3. 查看分析结果： 系统会高亮显示独立权利要求（蓝色）和从属权利要求（灰色）。
  4. 查看统计信息： 查看总权利要求数、独立权利要求数、从属权利要求数。
  5. 生成可视化图表： 选择可视化样式（树状图/网络图/径向图），生成权利要求引用关系图。
  6. 交互式操作： 使用缩放、拖拽、散开程度调节等功能查看图表细节。

可视化控制功能

  • 样式切换： 树状图（层次结构）、网络图（复杂关系）、径向图（中心辐射）
  • 散开程度调节： 树状图支持调节节点散开程度（0.5x-5.0x）
  • 缩放控制： 放大、缩小、重置、居中
  • 高清截图： 生成高清PNG图片保存

使用技巧

  • 对于大量专利权利要求的处理，建议使用Excel批量分析模式，效率更高。
  • 对于单条或少量权利要求的处理，建议使用文本输入分析模式，操作更便捷。
  • 使用树状图可以清晰展示权利要求的层次结构，适合查看引用层级。
  • 使用网络图可以直观展示权利要求之间的复杂引用关系。
  • 使用径向图可以从中心向外展示权利要求的引用关系，适合展示复杂的权利要求网络。
  • 通过公开号与独权合并视窗，可以快速了解每个公开号的核心技术内容。
  • 利用专利号查询功能，可以快速定位和可视化特定专利的权利要求结构。
  • 调节散开程度可以避免节点重叠，获得更清晰的可视化效果。

应用场景

  • 专利申请：帮助专利代理人快速分析和整理权利要求，确保权利要求结构合理。
  • 专利审查：帮助审查员快速理解专利的权利要求结构和引用关系。
  • 专利侵权分析：通过可视化引用关系，快速识别侵权风险点。
  • 技术分析：通过分析权利要求结构，了解技术的发展脉络和核心内容。
  • 专利数据库建设：将非结构化的权利要求文本转换为结构化数据，便于后续分析和检索。

功能八：专利附图标记

此功能用于自动识别专利附图中的标记数字，并与说明书中的标记说明进行智能匹配，生成带有标注的附图，支持多图批量处理和交互式查看。

核心技术优势

  • OCR智能识别： 采用先进的OCR技术，自动识别附图中的数字和字母标记。
  • AI辅助识别（可选）： 支持AI模式，利用大语言模型提升标记识别准确性，支持多语言说明书。
  • AI处理控制面板： 可配置AI模式开关、选择模型、自定义提示词，灵活控制识别过程。
  • 智能匹配算法： 自动将附图标记与说明书中的标记说明进行匹配，建立完整的标记映射关系。
  • 多图批量处理： 支持同时上传和处理多张附图，提高工作效率。
  • 交互式多图查看器： 提供专业的多图查看界面，支持缩放、拖拽、标记高亮等交互操作。
  • 图片旋转功能： 支持对上传的图片进行90度旋转（顺时针/逆时针），确保附图方向正确。
  • 快速重新处理： 支持单独重新识别OCR或重新匹配说明书，无需完整重新处理。
  • 缓存机制： 智能缓存处理结果，支持页面刷新后恢复，避免重复处理。
  • 实时进度显示： 处理过程中显示详细进度状态，让用户了解当前处理阶段。

AI处理控制面板

系统提供可选的AI处理控制功能：

  • AI模式开关： 开启后可使用大语言模型辅助识别
  • 模型选择： 选择适合的AI模型（如GLM-4-Flash等）
  • 自定义提示词： 可编辑AI识别提示词，优化识别效果
提示： AI模式需要消耗API额度，但可以获得更准确的识别结果，特别适用于复杂附图或多语言说明书。

工作流程

  1. 上传附图： 点击上传按钮选择专利附图文件，或直接粘贴图片。支持多张图片同时处理。
  2. 图片预处理： 如需调整图片方向，可使用旋转功能（↶逆时针/↷顺时针）对单张图片进行90度旋转。
  3. 输入说明书： 在文本框中输入或粘贴专利说明书内容，系统将从中提取标记说明。
  4. 配置AI模式（可选）： 如需使用AI辅助识别，在AI处理控制面板开启AI模式并选择合适的模型。
  5. 开始处理： 点击"开始处理"按钮，系统将自动进行OCR识别和说明书匹配。
  6. 查看结果： 处理完成后，查看识别统计信息（OCR识别数、说明书匹配数、平均置信度）。
  7. 交互式查看： 点击"打开多图查看器"进入交互式查看界面，详细查看每张附图的标记。

快速重新处理功能

当需要调整处理时，系统提供智能的重新处理选项：

  • 重新识别OCR（OCR按钮）： 当图片有旋转或需要重新识别标记时使用，会清除缓存重新进行OCR识别。
  • 重新匹配说明书（说明书按钮）： 当只修改了说明书内容时使用，会利用缓存的OCR结果快速重新匹配，速度提升50%以上。
提示： 系统会自动检测图片和说明书的变化。当图片发生变化（上传、删除、旋转）时，会提示清除缓存；当只有说明书变化时，可以快速重新匹配。

多图查看器功能

  • 缩略图导航： 左侧显示所有附图的缩略图，点击可快速切换。
  • 缩放与拖拽： 支持鼠标滚轮缩放和拖拽查看大图细节。
  • 标记高亮： 自动高亮显示识别到的标记，悬停可查看对应的说明书说明。
  • 标记列表： 右侧面板显示所有识别到的标记列表，点击可定位到对应位置。
  • 任务管理： 支持创建多个分析任务，方便对比不同批次的处理结果。

使用技巧

  • 上传图片前，建议先检查图片清晰度，清晰的图片能获得更好的OCR识别效果。
  • 如果附图方向不正确，先使用旋转功能调整方向再进行处理。
  • 对于复杂的附图或说明书包含多种语言的情况，建议开启AI模式以获得更好的识别效果。
  • 处理大量附图时，可以分批上传，系统支持多次添加图片。
  • 使用"保存缓存"功能可以在页面刷新后快速恢复之前的处理结果。
  • 在多图查看器中，可以使用鼠标滚轮缩放，按住鼠标拖拽查看大图细节。
  • 如果某些标记未能正确识别，可以尝试调整图片质量或切换到AI模式重新处理。
  • 修改说明书后，使用"重新匹配说明书"功能可以快速更新匹配结果，无需重新识别OCR。

应用场景

  • 专利审查：快速理解附图中的标记含义，提高审查效率。
  • 专利侵权分析：通过标记匹配，快速对比不同专利的附图技术特征。
  • 技术方案理解：帮助技术人员快速理解专利附图的技术细节。
  • 专利撰写辅助：检查附图标记与说明书的一致性，避免遗漏或错误。
  • 专利教学培训：生成带标注的附图，用于专利知识的教学和培训。
  • 专利数据库建设：将附图标记信息结构化，便于后续检索和分析。

常见问题排查

  • OCR未识别到标记： 检查图片清晰度，确保标记清晰可见；尝试调整图片方向；考虑使用AI模式。
  • 标记与说明书未匹配： 检查说明书内容是否完整；确认说明书语言与标记语言一致。
  • 处理超时： 对于大量图片或复杂图片，建议使用AI模式并耐心等待；或分批处理。

常见问题 (FAQ)

Q1: 为什么点击功能按钮后没有任何反应？

A: 最常见的原因是您没有设置API Key。请参照【入门指南】的第3步，正确配置并保存您的智谱AI API Key。如果已配置，请检查浏览器开发者工具(F12)的"控制台(Console)"和"网络(Network)"选项卡，查看是否有报错信息。

Q2: "小批量异步"和"大批量处理"有什么区别？我应该用哪个？

A:

  • 小批量异步： 适合几十到几百条任务，结果立等可取（几分钟到十几分钟）。任务在浏览器端轮询，关闭页面会中断（但可恢复）。
  • 大批量处理： 适合几千到几万条任务，是非实时的，通常需要几小时甚至一天。任务提交后完全在云端执行，您可以关闭浏览器。

简单来说，任务量小于500条，用"小批量异步"；大于500条，或需要长时间处理，用"大批量处理"。

Q3: 在"大批量处理"中，我不小心关闭了浏览器，任务还在吗？

A: 只要您已经成功点击"创建Batch任务"并获得了Batch ID，任务就在云端安全地执行。您只需重新打开页面，在"执行批处理"步骤的"断点续查"区域输入该Batch ID，即可恢复对任务状态的监控。

Q4: 上传文件失败或文件处理失败怎么办？

A: 请检查：

  1. 文件格式是否符合要求（如Excel文件应为.xlsx或.xls）。
  2. 文件内容是否正确，例如Excel文件的工作表(Sheet)中是否有数据。
  3. 网络连接是否正常。
  4. 如果问题持续，请联系管理员。
