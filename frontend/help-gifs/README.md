### 第三步：放入文件夹

将录制好的GIF文件放入 `frontend/help-gifs/` 文件夹中。

### 第四步：插入GIF（开发模式）

1. **开启开发模式**（二选一）：
   - 方式1：在浏览器地址栏添加 `?dev=1` 参数访问帮助页面
   - 方式2：在浏览器控制台执行 `enableDevMode()`

2. **打开GIF管理器**：
   - 点击右下角的 "🎬 GIF管理" 按钮
   - 或按 `Ctrl+G` 快捷键

3. **插入GIF**：
   - 在GIF管理面板中点击对应的GIF项
   - GIF会自动插入到对应的占位符位置

4. **验证效果**：
   - 刷新页面查看GIF是否正常显示
   - 检查GIF大小和清晰度是否合适

## 💡 注意事项

1. **文件大小**：建议单个GIF不超过 5MB，以保证加载速度
2. **尺寸建议**：宽度 600-800px，高度自适应
3. **帧率建议**：10-15fps，平衡流畅度和文件大小
4. **录制时长**：每个GIF控制在 10-30 秒，展示核心操作即可
5. **命名规范**：必须严格按照上表中的文件名命名，否则无法匹配

## 🔧 开发模式说明

### 开启开发模式

**方式1 - URL参数（临时）**：
```
http://localhost:8080/help.html?dev=1
```

**方式2 - 控制台命令（持久化）**：
```javascript
enableDevMode()
```

### 关闭开发模式

```javascript
disableDevMode()
```

### 开发模式功能

- 显示 "🎬 GIF管理" 按钮
- 支持 `Ctrl+G` 快捷键打开管理面板
- 可以插入和管理GIF文件

## 🚀 部署说明

部署到生产环境时：
1. 确保所有GIF文件已放入 `help-gifs/` 文件夹
2. 确保GIF管理按钮已隐藏（普通用户看不到）
3. 提交并推送代码到仓库

```bash
git add frontend/help-gifs/
git commit -m "docs: 添加帮助页面GIF演示文件"
git push origin main
```
