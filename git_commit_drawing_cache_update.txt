## 🔧 技术细节

### 缓存键生成
- 后端：MD5(图片内容) + 文件名
- 前端：简单哈希(图片数据) + 文件名
- 确保相同图片使用相同缓存

### 缓存过期策略
- 默认过期时间：7天
- 自动清理过期缓存
- 支持手动清除所有缓存

### API变更
**请求参数新增：**
```json
{
  "force_refresh": false  // 是否强制刷新缓存
}
```

**响应字段新增：**
```json
{
  "cache_info": {
    "drawing_name": {
      "has_cache": true,
      "cache_key": "drawing_hash123",
      "cached_at": 1704067200000
    }
  }
}
```
