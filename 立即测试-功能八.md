# 立即测试功能八 - 2步搞定

## 第1步：启动应用

打开终端，运行：

```bash
python run_app.py
```

**看到这个就对了：**
```
🚀 启动应用...
* Running on http://127.0.0.1:5001
```

**保持这个窗口打开！**

---

## 第2步：测试

**双击运行：** `测试认证绕过.bat`

**或者在新终端运行：**
```bash
python test_auth_bypass.py
```

---

## 看结果

### ✓ 成功

```
响应状态码: 200
✓ 成功！认证绕过工作正常
```

**→ 现在打开浏览器：** `http://localhost:5001/test_drawing_marker_debug.html`

---

### ✗ 失败

```
响应状态码: 401
✗ 失败！仍然需要认证
```

**→ 回到第1步的终端，找这些内容：**

```
[AUTH DEBUG] 请求路径: /api/drawing-marker/process
[AUTH DEBUG] ✓ 诊断工具请求，完全绕过认证
```

**如果看到 "✓ 诊断工具请求，完全绕过认证"：**
- 说明后端工作正常
- 问题在前端
- 截图发给我

**如果看到 "✗ 路径不匹配"：**
- 路径有问题
- 截图发给我

**如果没有看到 `[AUTH DEBUG]`：**
- 代码没更新
- 先运行：`Get-ChildItem -Path backend -Recurse -Filter "__pycache__" -Directory | Remove-Item -Recurse -Force`
- 然后重新执行第1步
