# 联网搜索功能 - README

## 🎯 功能简介

即时对话模块现已支持智谱AI网络搜索功能！AI可以实时获取最新的网络信息，让回答更准确、更及时。

## ⚡ 快速开始

### 3步启用联网搜索

1. **点击搜索按钮** 🔍（在输入框右侧）
2. **配置参数**（推荐使用默认配置）
3. **点击"保存并启用"**

就这么简单！

## 💡 何时使用

### ✅ 推荐使用场景
- 查询最新新闻和资讯
- 获取实时数据
- 了解最新技术发展
- 需要引用来源的专业问题

### ❌ 不需要使用场景
- 一般性知识问答
- 代码编写
- 文本分析
- 创意写作

## 🎨 功能特点

- ✅ **智能判断** - AI自动决定是否需要搜索
- ✅ **实时信息** - 获取最新网络资讯
- ✅ **来源标注** - 所有信息都有来源链接
- ✅ **多引擎支持** - 4种搜索引擎可选
- ✅ **灵活配置** - 可自定义搜索参数
- ✅ **清晰反馈** - 启用状态一目了然

## ❓ 常见问题

**Q: 如何知道AI使用了联网搜索？**  
A: AI的回答中会包含来源链接标注

**Q: 每次对话都会搜索吗？**  
A: 不会，AI会智能判断是否需要搜索

**Q: 如何关闭联网搜索？**  
A: 再次点击搜索按钮即可关闭

**Q: 搜索会产生费用吗？**  
A: 是的，按搜索次数计费（不是按对话次数）
