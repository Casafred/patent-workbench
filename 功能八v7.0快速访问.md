# 功能八v7.0 - 快速访问卡片 🚀

## 📍 两种访问方式

### 方式1：测试页面（推荐）

**最简单直接**，专门的调试测试页面：

```
http://你的服务器IP/test_interactive_marker_v7_debug.html
```

或使用域名：
```
http://ipx.asia/test_interactive_marker_v7_debug.html
```

**特点**：
- ✅ 完整的调试面板
- ✅ 独立测试环境
- ✅ 清晰的界面布局
- ✅ 详细的统计信息

---

### 方式2：主页面集成

在主系统中使用：

```
http://你的服务器IP/
```

**操作步骤**：
1. 登录系统
2. 进入"功能八：交互式附图标注"
3. 上传附图 + 输入说明书
4. 点击"开始处理"
5. **向下滚动** → 看到调试面板

**特点**：
- ✅ 集成在主系统中
- ✅ 无需切换页面
- ✅ 完整的功能体验
- ✅ 自动显示调试信息

## 🔍 调试面板功能

### 左侧：OCR识别结果
- 原始识别的所有标号
- 每个标号的位置和置信度
- 匹配状态（已匹配/未匹配）
- 🟢 绿色 = 已匹配
- 🟡 黄色 = 未匹配

### 右侧：说明书提取
- 说明书中的所有部件
- 标号和名称对应关系
- 识别状态（已识别/未识别）
- 🟢 绿色 = 已识别
- 🔴 红色 = 未识别

## 💡 使用提示

4. **对比两侧列表**：快速定位问题
