## 技术实现

### 语言检测规则
- 日语：检测平假名/片假名字符占比 > 5%
- 中文：检测汉字字符占比 > 10%
- 英文：检测英文字母占比 > 30%
- 德语：检测关键词 anspruch, gemäß, dadurch
- 法语：检测关键词 revendication, selon
- 韩语：检测韩文字符范围

### AI翻译流程
1. 前端检测到非中英文文本
2. 弹出对话框提示用户开启AI模式
3. 用户确认后发送请求到后端
4. 后端调用 glm-4-flash 模型翻译为中文
5. 使用正则规则解析翻译后的文本
6. 返回解析结果和翻译信息

### 翻译提示词优化
```
你是一个专业的专利文献翻译专家。请将以下{source_lang}文本翻译为中文。

要求:
1. 保持专利术语的准确性
2. 保留所有数字标记和权利要求序号
3. 翻译要流畅自然,符合中文专利文献的表达习惯
4. 不要添加任何解释或注释,只返回翻译结果
```
