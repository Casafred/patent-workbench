```bash

1. 上传专利附图

5. 体验所有新功能

```

## 💻 最简代码

```html
<script src="frontend/js/multiImageViewer_v8.js"></script>
<script>
const images = [{
    url: 'image.jpg',
    title: '图片1',
    detectedNumbers: [{ x: 100, y: 100, number: '1', name: '部件1' }],
    referenceMap: { '1': '部件1' }
}];

const viewer = new MultiImageViewerV8(images);
viewer.open(0);
</script>
```

## ✨ 六大功能

1. **调试面板** - 点击"🔧"按钮
2. **多图切换** - 鼠标移到边缘
3. **图片旋转** - 点击"↺↻"按钮
4. **选中高亮** - 点击标注列表
5. **功能栏** - 侧边栏集成
6. **手动标注** - 双击图片

## 🎮 快捷键

- `←` - 上一张图片
- `→` - 下一张图片
- `ESC` - 关闭查看器
- `滚轮` - 缩放图片
- `双击` - 添加标注
