## 新增功能：手动输入同族专利号

### 1. 修改HTML结构 (family-claims-comparison.html)
- 添加两个输入模式的切换标签页：
  - **自动获取**：输入一个专利号，自动爬取同族
  - **手动输入**：输入多个专利号，手动指定同族
- 手动输入模式添加：
  - textarea输入框（支持每行一个专利号或逗号分隔）
  - "添加到列表"按钮

### 2. 修改JavaScript逻辑 (familyClaimsComparison.js)
- 添加输入模式切换函数
- 添加解析手动输入专利号的函数
- 将手动输入的专利号添加到同族列表
- 支持混合模式：自动获取 + 手动补充

### 3. 后端API调整
- 复用现有的 `/api/patent/family/compare` API
- 无需新增API，前端直接将专利号列表发送给后端

### 界面布局
```
┌─────────────────────────────────────────┐
│  [自动获取]  [手动输入]                  │  ← 模式切换标签
├─────────────────────────────────────────┤
│ 自动获取模式：                           │
│   专利号：[__________] [获取同族列表]    │
├─────────────────────────────────────────┤
│ 手动输入模式：                           │
│   请输入专利号（每行一个或逗号分隔）：     │
│   ┌─────────────────────────────────┐   │
│   │ US1234567B2                     │   │
│   │ EP9876543A1                     │   │
│   │ CN202310000000A                 │   │
│   └─────────────────────────────────┘   │
│   [添加到列表]                          │
└─────────────────────────────────────────┘
```
