## 专利详情页双栏对照模式实现方案

### 一、功能概述

在专利详情新标签页中添加"双栏对照"浏览模式，允许用户点击切换按钮后，将页面分为左右两栏，每栏独立显示完整的专利详情内容，用户可分别滚动查看不同区域进行对照。

### 二、实现细节

#### 1. 添加切换按钮

* 在页面头部（`.header-top`）添加"双栏对照"切换按钮

* 按钮样式与现有PDF下载、Google Patents链接保持一致

* 点击切换时显示当前模式状态

#### 2. 双栏布局CSS设计

```css
/* 双栏模式容器 */
.dual-column-mode .container {
    display: flex;
    gap: 20px;
    max-width: none;
    padding: 0 20px;
}

/* 左右栏目 */
.dual-column-mode .column-left,
.dual-column-mode .column-right {
    flex: 1;
    max-width: 50%;
    background: white;
    border-radius: 16px;
    box-shadow: 0 20px 60px rgba(0,0,0,0.3);
    overflow: hidden;
    height: calc(100vh - 100px);
    overflow-y: auto;
}

/* 栏目标题标识 */
.column-header {
    position: sticky;
    top: 0;
    background: linear-gradient(135deg, #2e7d32 0%, #43a047 100%);
    color: white;
    padding: 10px 20px;
    font-weight: 600;
    z-index: 10;
}
```

#### 3. JavaScript切换逻辑

* 添加 `toggleDualColumnMode()` 函数

* 克隆内容区域到左右两栏

* 使用 `localStorage` 记住用户偏好

* 页面加载时恢复上次的模式设置

#### 4. 用户体验优化

* **独立滚动**：每栏可独立滚动，方便对照不同区域

* **同步滚动开关**：提供可选的同步滚动功能

* **分隔线**：两栏之间有明显的视觉分隔

* **响应式处理**：窗口宽度小于1200px时提示并自动切回单栏

* **导航栏适配**：左侧导航栏在双栏模式下保持可用

### 三、修改文件

* `js/patentDetailNewTab.js` - 添加切换按钮HTML、CSS样式和JavaScript逻辑
