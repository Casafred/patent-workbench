## 现代科技风格中文首页设计方案

### 一、创建新文件
创建 `frontend/landing.html` 作为新的首页

### 二、页面结构设计

#### 1. 导航栏（固定顶部）
- 左侧：Logo + 品牌名称 "ALFRED X IP"
- 右侧：功能导航链接 + **登录按钮**（点击跳转到 `/admin_manage`）

#### 2. Hero 区域（首屏视觉冲击）
- 大标题：动态打字机效果展示核心价值
- 副标题：产品定位说明
- CTA按钮：立即体验 / 了解更多
- **酷炫渐变背景**：使用功能截图 + 渐变叠加层实现

#### 3. 功能特性展示区
- 参考 help.html 的功能模块组织
- 九大核心功能卡片展示
- 每个功能卡片预留截图/动画位置

#### 4. 图片演示区
- 功能截图轮播/网格展示
- 提供详细的图片样式设置教程注释

#### 5. 页脚
- 版权信息、联系方式、快速链接

### 三、视觉效果实现

1. **渐变背景图片功能**
   - CSS `background-image` + `linear-gradient` 叠加
   - 支持自定义截图作为背景
   - 提供配置教程

2. **动画效果**
   - Vanta.js 粒子背景（与现有风格一致）
   - 滚动视差效果
   - 元素入场动画

3. **响应式设计**
   - 移动端适配
   - 平板适配

### 四、技术实现

- 使用项目现有的 CSS 变量系统（绿色主题）
- 复用 Vanta.js 背景动画
- 纯 HTML/CSS/JS 实现，无需额外依赖
