## 计划：添加PaddleOCR-VL-1.5作为第三种OCR选项

**步骤1.1** 创建PaddleOCR-VL测试脚本
- 编写测试脚本调用PaddleOCR-VL-1.5 API
- 使用 `test_data/test ocr.png` 进行测试
- 验证返回结果中是否包含文本坐标（`prunedResult`结构）

**步骤2.3** 添加配置支持
